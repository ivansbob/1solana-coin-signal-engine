# Drift Perp Context

Optional context tracking providing explicit dependencies verifying boundaries structurally natively bridging execution limits logically mapping risk behaviors naturally without over-scaling outputs.

## Contextual Modifiers Only
Unlike Liquidity slippages (`execution_grade`) ensuring physical blocks, `PerpContext` metrics act strictly as **Modifiers**. They evaluate Open Interest structures verifying boundaries matching parameters safely providing explicitly safe +0.05 bonus additions natively bounding confidence limits dynamically without overriding explicit hard checks logically.

## Graceful Degradation
Whenever `drift_context_status` fails routing `missing`/`stale` outputs, confidence modifier physically plummets down cleanly outputting 0.0 explicitly ensuring executions never generate "False Positive confidence" over blanks inherently matching exact paper setups.
